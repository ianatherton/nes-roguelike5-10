Quadrant Arrangement Explanation

In NES games, sprites are often composed of multiple 8x8 tiles arranged in a grid. The standard arrangement is row-major order [0, 1, 2, 3], but Arkista's Ring uses a different arrangement where the top-right and bottom-left quadrants are swapped: [0, 2, 1, 3]

Standard Arrangement [0, 1, 2, 3]:

┌───┬───┐
│ 0 │ 1 │
├───┼───┤
│ 2 │ 3 │
└───┴───┘

Arkista's Ring Arrangement [0, 2, 1, 3]:

┌───┬───┐
│ 0 │ 2 │
├───┼───┤
│ 1 │ 3 │
└───┴───┘
